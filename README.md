Fast Food Nutrition Analysis — Power BI Project

## 📌 Project Overview
Analysis of nutritional characteristics of fast-food items from multiple restaurant chains to identify healthier choices and high-calorie or high-sodium items using Power BI dashboards.

## 🛠️ Tools Used
- Power BI (Dashboards, DAX)
- SQL (Data extraction & analysis)

## 📊 Dataset
- *10+ restaurant chains* analysed (McDonald's, Burger King, Subway, Pizza Hut, KFC, etc.)
- *17 nutritional attributes* — Calories, Fat, Sodium, Protein, Vitamins, Calcium, etc.

## 📈 Dashboards Built
| Dashboard | Description |
|-----------|-------------|
| Overall Analysis | Restaurant-wide nutritional comparison across all chains |
| McDonald's Analysis | Deep-dive into McDonald's 57+ menu items |

## 💡 Key Insights
- *McDonald's avg calories: 640.35* with *61.4% high-calorie items*
- *Avg sodium across restaurants: 1.44K mg* — posing health risks
- *Subway* provides healthiest alternatives with lower calories & sodium
- *Pizza Hut & KFC* still have high sodium levels
- High calories strongly correlate with high sodium in McDonald's items
- Lowest calorie options: small chicken servings, wraps, and sides

## 🎯 Project Objectives
- Analyse nutritional info (calories, fat, sodium, protein) across restaurants
- Compare restaurant-wise food health impact
- Identify high-risk items based on calorie and sodium levels
- Create interactive dashboards for quick insights
